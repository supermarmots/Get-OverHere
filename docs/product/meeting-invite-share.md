# 약속 초대 링크 공유 화면 기획

## 목적

약속 생성 직후 주최자가 참여자에게 공유할 초대 링크를 확인하고 복사할 수 있게 한다. 생성 직후 바로 대시보드로 돌아가면 공유 행동이 끊기므로 공유 전용 화면을 먼저 보여준다.

## 현재 사용자 흐름

1. 주최자가 약속 생성 마지막 단계에서 생성을 완료한다.
2. `/meetings/{meetingId}/invite`로 이동한다.
3. 약속 제목과 초대 링크를 확인한다.
4. `초대 링크 복사`를 누른다.
5. 성공/실패 메시지를 확인한다.
6. `대시보드로 이동`을 눌러 돌아간다.

## 링크 정책

```text
{origin}/meetings/{meetingId}/join
```

- 참여자는 이 링크로 `/meetings/{meetingId}/join`에 접근한다.
- 비로그인 상태면 로그인/회원가입 후 원래 join route로 돌아온다.
- 대시보드의 `초대 참여` dialog도 같은 경로만 허용한다.

## UI 원칙

- 다크 테마 유지
- 링크가 화면 밖으로 넘치지 않도록 줄바꿈 처리
- Clipboard API 실패 시 직접 선택/복사할 수 있게 링크 원문 표시
- `main`, `section`, `header`, `footer`, `p`, `button` 중심의 시맨틱 구조 사용
