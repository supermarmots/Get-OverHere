# 약속 초대 링크 공유 화면 기획

## 목적

약속 생성 직후 주최자가 참여자에게 공유할 초대 링크를 확인하고 복사할 수 있는 화면이다. 약속을 만든 뒤 바로 대시보드로 돌아가면 공유 행동이 끊기므로, 생성 성공 후 공유 전용 화면을 먼저 보여준다.

## 사용자 흐름

1. 주최자가 `약속 만들기` 마지막 단계에서 생성을 완료한다.
2. `/meetings/{meetingId}/invite` 화면으로 이동한다.
3. 생성된 약속 제목과 초대 링크를 확인한다.
4. `초대 링크 복사`를 누른다.
5. 복사 성공 메시지를 확인한다.
6. `대시보드로 이동`을 눌러 돌아간다.

## 화면 구성

### 상단

- 서비스명 또는 `약속 생성 완료`
- 짧은 안내 문구: `이 링크를 참여자에게 공유해 주세요.`

### 본문

- 약속 제목
- 초대 링크
- 복사 버튼

### 하단 액션

- `초대 링크 복사`
- `대시보드로 이동`

## 링크 정책

1차 구현에서는 다음 형식을 사용한다.

```text
{origin}/meetings/{meetingId}/join
```

아직 참여 페이지가 없더라도 링크 형식은 미리 고정한다. 참여 페이지 구현 시 동일 경로를 사용한다.
